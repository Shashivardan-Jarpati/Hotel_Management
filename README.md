# Hotel Management System (Python)

## Project Overview
A console-based Hotel Management System developed using Python and Object-Oriented Programming concepts.  
The system manages room allocation, guest check-in/check-out, and room availability efficiently.

## Features
- Add and manage different room types with pricing
- Guest check-in and check-out system
- Real-time room availability tracking
- Guest information retrieval
- Menu-driven console interface

## Technologies Used
- Python
- Object-Oriented Programming (OOP)

## How to Run
1. Ensure Python is installed on your system
2. Run the file using:
   python hotel_management.py

## Use Case
This project demonstrates practical usage of Python OOP concepts in building a real-world management system.
